NFTDocs

NFTDocs is an Dapp that lets institutions securely issue immutable marksheets to the students along with the digitally signed marksheet in the form of NFT. It also adds a CGPA claim to the student's Polygon ID wallet claiming the CGPA that student received.

Further On Chain Verification by Polygon ID is done to verify the CGPA details and Marksheets which after successful Verification issues verification claim to students' Polygon ID wallet making student eligible for placement rounds.

Project was build using Ethereum, Polygon-ID, Push, NFTPort, Solidity, MetaMask and ReactJS
